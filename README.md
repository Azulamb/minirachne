# Minirachne

Minirachne is a Web API server made with Deno.

## Need

- `Deno ^v1.15`
  - Use [URLPattern](https://developer.mozilla.org/en-US/docs/Web/API/URLPattern).

## Sample

See `sample/server.ts` .

## Command

* `deno fmt`
  * Format.
* `./cache.sh`
  * Cache clear.
